Qué datos se tratan

Esta web no tiene backend, base de datos ni formulario conectado a servidor propio. Si contactas por WhatsApp, teléfono, email o Instagram, se tratarán los datos que decidas enviar para responder a tu consulta.

Finalidad

Los datos se usan para responder solicitudes de información, valorar un posible proyecto, preparar una propuesta y mantener la comunicación necesaria si decides contratar el servicio.

Base legal

La base del tratamiento es tu solicitud de información y, en su caso, la relación precontractual o contractual necesaria para preparar y ejecutar el encargo.

Conservación

Los mensajes y datos de contacto se conservarán durante el tiempo necesario para gestionar la consulta o el proyecto, y después durante los plazos exigibles por obligaciones legales si aplica.

Cesiones y terceros

No se venden datos personales ni se ceden a terceros para campañas comerciales. Los canales externos como WhatsApp, email, teléfono o Instagram tienen sus propias condiciones y políticas de privacidad.
