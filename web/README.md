# CreadorWebs

Esta carpeta contiene la web pública de `https://creadorwebs.es`.

Netlify debe publicar únicamente `web/`.

## Contenido

- `index.html`: página principal publicada.
- `styles.css`: estilos principales.
- `script.js`: navegación, modales, carrusel y visor de demos.
- `assets/`: favicon e imágenes del portfolio.
- `privacidad.html`: página de privacidad.
- `cookies.html`: página de cookies.
- `demos/`: demos ficticias mostradas como ejemplos visuales.

## Demos

Las demos están dentro de `web/demos/` y representan ejemplos ficticios para distintos tipos de proyecto.

Todas las demos deben mantenerse con `noindex, nofollow` para evitar que se indexen como negocios reales.
