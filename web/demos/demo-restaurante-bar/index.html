FAQ

Preguntas habituales

¿Se puede reservar online?

En esta demo el botón solo muestra un aviso. En una web real se conectaría con WhatsApp, teléfono o reserva.

¿La carta es real?

No. Los platos son contenido ficticio para enseñar estructura y presentación.

¿Sirve para bares pequeños?

Sí. Se puede simplificar para menú diario, tapas, desayunos o copas.
