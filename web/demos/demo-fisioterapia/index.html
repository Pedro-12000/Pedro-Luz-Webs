FAQ

Preguntas habituales antes de pedir cita.

¿Necesito diagnóstico médico?

Depende del caso. Si ya tienes informes o pruebas, puedes traerlos. Si aparecen señales que requieren valoración médica, se te orientaría para consultarlo con el profesional adecuado.

¿Cuánto dura una sesión?

La duración habitual se acordaría según el tipo de sesión. Como referencia orientativa, muchas sesiones de fisioterapia se sitúan entre 45 y 60 minutos.

¿Puedo pedir cita por WhatsApp?

Sí. En esta demo el botón muestra un aviso interno. En una web real se configuraría con el número del centro.

¿Trabajáis con seguros?

Esta página no representa acuerdos con aseguradoras. En un proyecto real se indicarían solo los seguros o condiciones confirmadas por el centro.

¿La primera visita es gratuita?

La primera valoración orientativa puede variar según cada centro. En una web real se explicaría con claridad si tiene coste, duración y condiciones.
